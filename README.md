## Featured work

A selection of apps I’ve built across mobile, AI, and full-stack product systems. Most are pre-production builds focused on product exploration, UX, and architecture.

### FrizzCheck

A weather-driven mobile app that helps users anticipate how humidity and forecast conditions may affect their hair so they can make faster styling decisions.

**Stack:** React Native, Expo, React Navigation, React Native Paper, Axios, OpenWeather API

### SayHey

A language utility app for quick translation workflows, phrase capture, and everyday communication support in a fast mobile interface.

**Stack:** React Native, Expo, JavaScript, AsyncStorage, Axios, Google Translate API integrations

### RudeBoi

A personality-driven AI mobile coach focused on conversational UX, archives, citations, and subscription-aware product flows.

**Stack:** React Native, Expo, TypeScript, MobX, expo-router, WebView, Google Sign-In

### ShelfAware

A grocery-scanning app that uses OCR and AI-assisted analysis to evaluate shelf products against nutrition goals, ingredient preferences, and dietary constraints.

**Stack:** React Native, Expo, MobX, Node.js, Express, Supabase, Google Vision, Perplexity, RevenueCat

### ShotTracker

A basketball training app for solo shooting sessions with calibration, camera-assisted tracking, and progress history over time.

**Stack:** React Native, Expo, TypeScript, vision-camera, Zustand, expo-sqlite, Supabase

### SportsGuyAI

A sports-focused conversational AI assistant designed for matchup and performance exploration with citation-backed answers and repeat-use chat flows.

**Stack:** React Native, Expo, TypeScript, MobX, RevenueCat, Google Sign-In, WebView, Node.js APIs

### Twogether

An iOS-first app for couples to schedule and protect shared phone-free time, with pairing flows and Screen Time / Family Controls integration.

**Stack:** React Native, Expo, TypeScript, Expo Router, Zustand, TanStack Query, RevenueCat, Supabase, Swift module integration

### DreamCapture

An iOS-first dream journaling experience that turns entries into AI-generated visuals and lets users revisit memories through a calendar view.

**Stack:** React Native, Expo, TypeScript, Node.js, Express, MongoDB, Firebase Storage, RevenueCat

### Attune

A mindful journaling and reflection platform combining daily check-ins, AI-guided coaching (“Genie”), and long-term insight generation.

**Stack:** React Native, Expo, TypeScript, Node.js, Express, MongoDB, Redis, BullMQ, OpenAI, Anthropic

### MediCount

A medication-counting mobile product that uses guided capture plus AI-assisted image processing to reduce manual counting effort.

**Stack:** React Native, Expo, TypeScript, MobX, Firebase Storage, AI image-analysis workflow

### peaq

An AI-powered health coaching platform built around a closed feedback loop: collect signals, adapt plans, trigger the right interventions, and track adherence.

**Stack:** TypeScript, React Native, Node.js, PostgreSQL, Supabase, Zod, pnpm workspaces, Turbo

### PSS

A private-security mobile experience for client onboarding, verification, and dashboard access, built with a premium visual style.

**Stack:** React Native, Expo, TypeScript, MobX, React Navigation, Shopify Skia, expo-secure-store
